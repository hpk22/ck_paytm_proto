# Paytm × CashKaro — Web Journey Prototype

Interactive, clickable product prototype (the "proto") for the **Paytm × CashKaro Cashback & Offers (CBO)** partnership pitch. This is the live product walkthrough — Shop → Track → Earn — that is embedded as the demo inside the pitch deck.

## Live demo
Hosted on GitHub Pages: **https://hpk22.github.io/ck_paytm_proto/**

## What's here
- `index.html` — the entire prototype as a single, self-contained HTML file (exported from Claude Design). React is loaded at runtime from the unpkg CDN, so the page needs an internet connection to run.
- `.nojekyll` — tells GitHub Pages to serve the files as-is (no Jekyll processing).

Only the prototype is deployed here. The pitch deck itself is kept separate.
